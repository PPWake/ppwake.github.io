WWDC 2019

Jun 4, 2019 | 阅读

第一次和小伙伴组队看 wwdc 2019，就着可乐烧烤，畅谈技术，碰撞想法，一起嗨皮！

本届 WWDC 对开发者来说可谓干货满满，下面逐一列举：

SwiftUI

这很 swift！

声明式，响应式，让 UI 编程更简单，进一步降低入门难度，有点像 flutter。

ARKit 3

我很酷！

  • RealityKit
  • 能够识别人体和肢体动作

隐私与安全

你的数据，你做主！

定位权限

  • App 再次使时定位权限时，需要重新授权
  • 进一步，限制 app 通过扫描 WIFI 和 蓝牙 来定位用户

社交帐号

  • 使用社交帐号登录其他 App (例如微信登录) 会泄露大量的个人信息 (年龄，性别，邮箱，姓名，头像）

    现在可以使用使用 Apple 的帐号系统来授权其他 App，并且可以控制共享哪些信息项，还能随机生成匿名邮箱。

    非好棒的功能，amazing！

iOS

性能杠杠的！

  • Face ID 解锁速度提高 30%
  • 应用启动速度提升 200%
  • App 安装包缩小一半

iPad OS

不止娱乐！

  • 可插 U 盘

  • 多屏显示

  • 三指新手势，例如：三指捏复制，三指滑动撤销

Watch OS

我很独立！

  • 独立 App Store

  • 独立 App

Mac OS

更丰富！

  • Project Catalyst - iPad 应用可以轻松移植到 Mac

  • Sidecar - 把 iPad 做为第二显示屏

买不起系列

  1. iMac pro - 低配 4 万
  2. 显示器 - 3万
  3. 支架 - 6千
